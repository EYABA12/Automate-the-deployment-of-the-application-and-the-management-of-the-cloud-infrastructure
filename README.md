# Projet de STAGE d’été: SECOPS 360

## À propos du projet
Automatisation des déploiements pour une application appelée **Minitrello**, réalisée avec **NodeJS**, **React** et **MongoDB**.  
**About the project:** Automation of deployments for an application called **Minitrello**, developed with **NodeJS**, **React**, and **MongoDB**.

## Cadre du projet
- **Dockerisation de l'application Minitrello**  
  **Containerization of the Minitrello application**
- **Automatisation du déploiement avec Jenkins**  
  **Automation of deployment with Jenkins**

- **Gestion des requêtes et du trafic avec Nginx Proxy Manager**  
  **Management of requests and traffic with Nginx Proxy Manager**
- **Adoption de l'Infrastructure as Code (IaC)**  
  **Adoption of Infrastructure as Code (IaC)**
- **Mise en place d'une plateforme d'automatisation IT avec Ansible**  
  **Establishment of an IT automation platform with Ansible**
- **Déploiement de l'application sur une machine virtuelle Azure**  
  **Deployment of the application on an Azure virtual machine**
- **Utilisation de la méthodologie Agile Scrum pour la gestion de projet**  
  **Use of Agile Scrum methodology for project management**
